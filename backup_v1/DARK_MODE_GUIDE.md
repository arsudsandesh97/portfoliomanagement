- Clear browser cache and try again

### Colors Look Wrong?
- Try toggling the theme twice
- Refresh the page
- Check browser compatibility (Chrome 90+, Firefox 88+, Safari 14+)

- The theme transition is designed to be smooth
- If you experience lag, try closing other browser tabs
- Ensure browser is up to date
